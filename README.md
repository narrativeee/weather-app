### Weather App | Minimal

Минималистичное веб-приложение для мониторинга погоды в реальном времени. Интерфейс выполнен в строгом монохромном стиле с акцентом на типографику и удобство пользователя.

[Живое демо (GitHub)](https://narrativeee.github.io/weather-app)  
[Зеркало на Netlify](https://weather-app-narrativiii.netlify.app)

### Особенности:
* **Real-time Data:** Актуальная информация о температуре, скорости ветра и погодных условиях.
* **Geocoding Search:** Быстрый поиск координат и данных для любого города мира.
* **Search History:** Интеллектуальная панель быстрого доступа к 3 последним поисковым запросам.
* **Smart UI:** Возможность мгновенной очистки истории и полная адаптация под мобильные устройства.
* **Dark Mode:** Контрастный темный дизайн, минимизирующий нагрузку на зрение.

### Технологии:
* **HTML5** - семантическая структура и современная разметка.
* **CSS3** - кастомная верстка, использование Flexbox и анимаций появления (`Keyframes`).
* **JS (Vanilla)** - асинхронная логика приложения, глубокая работа с DOM и обработка событий.
* **REST API (Fetch):** Интеграция с сервисом `Open-Meteo` для получения метеоданных в формате JSON.
* **LocalStorage API:** Сохранение состояния истории поиска между сессиями браузера.
