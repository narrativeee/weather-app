### Weather App

Минималистичное веб-приложение для мониторинга погоды в реальном времени. Интерфейс выполнен в строгом монохромном стиле с акцентом на типографику и удобство пользователя.

[Живое демо](https://narrativeee.github.io/weather-app)  
[Зеркало на Netlify](https://weather-app-narrativiii.netlify.app)

### Особенности:
* Real-time Data: Актуальная информация о температуре и погодных условиях.
* Geocoding: Поиск координат любого города через внешние сервисы.
* Search History: Панель быстрого доступа к 3 последним городам.
* Smart UI: Возможность очистки истории и адаптация под мобильные устройства.
* Dark Mode: Контрастный темный дизайн, комфортный для глаз в любое время суток.

### Технологии:
**HTML5** — семантическая структура.

**CSS3** — современная верстка, Flexbox, анимации появления (Keyframes).

**JS (Vanilla)** — асинхронная логика, работа с DOM и обработка событий.

**REST API (Fetch)** — взаимодействие с Open-Meteo для получения геопозиции и погодных данных.

**LocalStorage** — хранение истории поиска между сессиями браузера.
